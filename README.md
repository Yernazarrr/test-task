# Тестовое задание

Полноценное Flutter приложение для авторизации через email с использованием JWT токенов.

**Особенности**:
  - Clean Architecture
  - BLoC State Management
  - JWT + Refresh Token
  = Локализация (EN/RU)
  - Dependency Injection (GetIt)
  - Навигация (GoRouter)
  - Валидация форм
